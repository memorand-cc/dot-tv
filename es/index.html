¿Qué es Portobello?

Portobello es un complemento para Stremio.

¿Qué es Stremio?

Stremio es una app para reproducir vídeos en streaming.

¿Cómo funciona Portobello?

Del mismo modo que una aplicación de reproducción de vídeo normal en tu dispositivo necesita que le proporciones un archivo de vídeo, Stremio es una aplicación de reproducción de vídeo en streaming que necesita una fuente de streaming para funcionar.

Portobello rastrea contenidos de acceso público en Internet y los envía a la aplicación del reproductor Stremio como fuente de streaming. Actúa como intermediario entre el contenido y el reproductor, permitiendo que este último pueda interpretar el contenido.

Ni Stremio ni Portobello alojan ningún contenido. Stremio es simplemente un reproductor de vídeo en streaming de código abierto que permite ampliar su funcionalidad mediante complementos de software. Portobello es sólo un complemento que indexa, recopila y organiza contenidos de dominio público disponibles en ubicaciones ajenas.

¿Qué relación hay entre Stremio y Portobello?

Stremio y Portobello no estan relacionados en forma alguna. Portobello es un complemento para Stremio, del mismo modo que Grammarly es una extensión para Chrome. Se trata de dos programas que funcionan juntos, pero que han sido desarrollados por dos entidades diferentes que no mantienen relación alguna entre sí.

¿Qué datos personales se almacenan?

Los únicos datos que se conservan son los utilizados para rellenar el formulario de este sitio web. Nada más.

Stremio guarda tu dirección de correo electrónico y tu contraseña para que puedas iniciar sesión, mientras que Portobello solo conserva tu dirección de correo electrónico para informarte de cualquier problema técnico que surja durante la fase beta y cuándo finalizará esta. Jamás se enviará spam.

No se recopilan datos de ningún tipo sobre el uso del servicio. Portobello está diseñado intencionalmente para no observar ni guardar absolutamente ningún dato sobre su uso.

¿Necesito una VPN?

No. El uso de una VPN puede ser una buena práctica por motivos de seguridad y privacidad, pero no es necesaria para utilizar Portobello, ya que este se basa en tráfico de streaming http estándar.

¿En qué idiomas está disponible el contenido?

Prácticamente todo el contenido disponible públicamente se encuentra en versión original. Algunos archivos incluyen varias pistas de audio en otros idiomas, pero es muy posible que este no sea el caso del contenido concreto que desees ver.

Algunos archivos incluyen subtítulos incrustados en varios idiomas, y Portobello integra automáticamente los subtítulos de Opensubtitles para cada archivo en todos los idiomas disponibles.

¿Por qué se ven mal los colores en algunos vídeos?

Estás reproduciendo un archivo con un perfil HDR o Dolby Vision que no es compatible con tu dispositivo. Selecciona otro archivo.

¿Por qué se oye mal el sonido en algunos vídeos?

Estás reproduciendo un archivo con un perfil de sonido envolvente que no es compatible con tu dispositivo. Si es posible, elige otra pista de audio. Si no, selecciona otro archivo.

¿Por qué se interrumpen algunos vídeos?

Ten en cuenta la velocidad de tu conexión a Internet y el tamaño del archivo que deseas reproducir. Para hacerlo fácil, Portobello muestra primero los archivos seleccionados por su equilibrio entre calidad y tamaño (y los marca con una estrella), pero si bajas en la lista para elegir un archivo de vídeo 4K HDR de 40 GB, asegúrate de disponer de una conexión a Internet de banda ancha estable y lo suficientemente rápida como para reproducirlo.
